Flow Designer

The DESIGNER turns a flow into a polished, client-facing walkthrough. Build it from scratch, or Import a flow the Maker built — same xFLOW schema.

  • Edit the title, intro, and steps. Each step has a title, the prompt/action, a description, and media.
  • Add media per step — upload images or paste an image/video/embed link.
  • Preview shows the client view with the version badge; zoom with the HUD.
  • Publish sets the version label and share message; Export HTML writes a self-contained deliverable.
  • Export JSON writes the shared schema (graph preserved) so the flow round-trips back to the Maker.
